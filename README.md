# A-lightweight-responsive-static-website-HTML-CSS-JavaScript-.
This repo contains a single-page static site built with semantic HTML, modular CSS, and vanilla JS for interactions. The site is optimized for performance and accessibility — no frameworks, no bundlers. Use it as a portfolio, product landing page, or marketing prototype. Deploy easily via GitHub Pages, Netlify, or any static host.
Features
Mobile-first responsive layout
Semantic HTML and accessible structure
Smooth scroll and simple JS-driven interactions
Minimal, optimised assets for fast load times
Easy to customise — replace content and images
Tech stack
HTML5
CSS3 (Flexbox / Grid)
JavaScript (ES6)
Optional: live-server for local dev
Installation & local dev
Clone the repo
git clone https://github.com/<your-username>/<repo-name>.git
Open index.html in your browser — or run a dev server:
npx http-server OR npx live-server
Edit files in the root (index.html, styles.css, app.js)
Deployment
GitHub Pages: push to the gh-pages branch or enable Pages from main.
Netlify/Vercel: drag & drop or point to repo (no build command).
Contributing
PRs welcome. For UI changes, include before/after screenshots and test on mobile + desktop.
